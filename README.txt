  main.py and dicestimulator.kv is attached.
  dice stimulator using console.py is different file.

For main.py and dicestimulator.kv:
1. Create your own project
2. Copy the files to that project location
3. Run the main.py file

Note:
After creating the project, you have to install some Kivy packages to the interpreter
Go to, File>Setting>Python Interpreter
Click on (+) button
Search, Kivy
Install the below packages:
Kivy
kivy-deps.angle
kivy-deps.glew
kivy-deps.gstreamer
kivy-deps.sdl2

For dice stimulator using console.py:
Just copy and run the program.
